# Odd or even
## Statement
```
Given an integer, print "odd" if it's odd and print "even" otherwise.
```
## Example input
```
5
```
## Example output 
```
odd
```
## Example input 
```
6
```
## Example output 
```
even
```
```
a = int(input())
if(a%2==0):
  print("even")
else:
  print("odd")
```
